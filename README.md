# Computer Repair Management System

## Project Overview

The Computer Repair Management System is a software solution designed to help computer repair shops manage customer devices, track repairs, and record service details efficiently.

The system allows technicians to register customer devices, record accessories received with devices (such as chargers or bags), track repair progress, and manage repair pricing and profit.

## Documentation

* [System Specification](SPECIFICATION.md)
* [System Architecture](ARCHITECTURE.md)

## Planned Features

* Customer registration
* Device intake recording
* Repair status tracking
* Accessory tracking (charger, bag)
* Service handling options (pickup or delivery)
* Internal or external repair tracking
* Repair pricing and profit calculation

## Technology Stack

* Rust
* Ubuntu Linux
* VS Code
* SQLite (planned database)
